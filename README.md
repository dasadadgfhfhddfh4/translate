Современный веб-переводчик на FastAPI с минималистичным тёмным дизайном. Поддерживает 100+ языков через Google Translate.

✨ Возможности
🎨 Современный UI — тёмная тема с плавными анимациями
🌍 100+ языков — автоматическое определение исходного языка
⚡ Быстрый — работает через REST API
📱 Адаптивный — отлично работает на мобильных устройствах
🔒 Валидация — Pydantic для проверки входных данных
🚀 Лёгкий деплой — готов к хостингу на Render/Railway

🐛 Возможные ошибки
Ошибка перевода: ...
Проверь подключение к интернету
Google Translate может временно блокировать запросы
Попробуй другой текст или язык
Ошибка соединения с сервером
Убедись, что сервер запущен
Проверь, что порт 8000 открыт (для локальной сети)
На Render не запускается
Проверь логи в панели Render
Убедись, что requirements.txt содержит все зависимости
Используй --host 0.0.0.0 в команде запуска
